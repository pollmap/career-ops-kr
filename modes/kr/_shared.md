> 이 파일은 `modes/kr/*` 모든 한국 특화 모드(`scan`, `filter`, `score`, `auto-pipeline`)가 상속하는
> 한국 금융권 고유의 용어/관행/법규 레이어다.
> 디폴트 영어 모드(`modes/*.md`)는 일반 분류/스코어링을 담당하고, 이 레이어는 그 위에 한국어 JD에만
> 적용되는 추가 규칙을 얹는다.

## 한국 금융권 용어 상속

- **용어집 포인터**: `modes/kr/glossary.md` — 180+ 엔트리 (급여/근로/직급/자격증/조직/공공/법규/핀테크)
- 모든 `modes/kr/*` 모드는 텍스트 파싱 시 glossary 엔트리를 1차 매칭 사전으로 사용한다.
- glossary에 없는 신규 용어가 반복 등장하면 → patterns 모드가 벡터에 저장 → 다음 스캔에서 glossary 후보 제안.
- 용어 정의와 실 본문 문구 해석이 충돌하면 **공고 원문이 우선**. glossary는 해석 힌트일 뿐이다.

## 호봉/직급 체계

한국 금융권 위계(은행/증권/자산운용 공통, 보직은 별도):

```
사원(1~4호봉)  →  대리(5~8호봉)  →  과장(9~13호봉)  →  차장(14~17호봉)  →  부장(18~23호봉)
                                                                              │
                                    ┌─────────────────────────────────────────┘
                                    ↓
                 팀장/본부장(보직) → 이사대우/이사 → 상무 → 전무 → 부사장 → 사장/대표
```

- **호봉(號俸)**: 근속연수별 기본급 테이블. 은행은 보통 1~23호봉, 증권은 더 압축.
- **보직(補職)**: 팀장/본부장/센터장은 직급과 별개로 부여되는 관리 책임. "과장(팀장)" 조합 가능.
- **정규직 전환 패턴이 관찰되는 직급**: 사원(인턴→정규) / 대리(경력직 입사) / 과장(헤드헌팅 스카우트)
- 찬희는 **신입=사원급** 타겟. 공고에 "대리급 이상" 나오면 자동 FAIL.

## 급여 구조

한국 금융권 연봉은 단일 숫자로 오는 경우가 드물다. 반드시 구성요소를 분해한다:

```
연봉 총액 = 기본급(×12 또는 ×13) + 제수당 + 상여금(성과·명절) + 복리후생비
```

- **기본급**: 호봉표 고정. 월 200~400만원대 사원~과장.
- **제수당**: 직책수당, 근속수당, 연장근로수당, 야간수당, 휴일수당, 자격수당.
- **상여금**: 명절상여(설·추석), 성과급(PS/PI — Profit Sharing / Profit Incentive). 증권사는 PS 변동 큼.
- **복리후생비**: 식대(비과세 월 20만원 한도), 교통비, 경조사비, 자녀학자금, 건강검진, 콘도·리조트.
- **실수령액 vs 원천징수**: 세전(원천징수영수증 기준) ≠ 세후(실수령액). 공고는 대부분 세전.
- **퇴직금**: DB(확정급여형) / DC(확정기여형) / IRP(개인형) 중 하나. 증권사는 DC 일반적.

**파서 규칙 (kr/score.md에서 사용)**:
- "연봉 4800" / "연 4,800만원" / "4800만원" → 48,000,000
- "월 300" / "월 300만원" → ×12 + 성과급 가정 15% → 41,400,000
- "기본급 3000 + 성과급 별도" → 30,000,000 (성과급은 별도 필드)
- "초봉 기준" / "신입 기준" → 확실한 하한선
- 범위 표기("4000~5000") → 중간값 + 주의 플래그

## 정규직 전환 패턴

```
[체험형 인턴 3~6개월]     → 전환 없음. 경력 스펙만 쌓기. (예: 한국은행 체험형)
[채용형 인턴 2~6개월]     → 평가 통과 시 정규직 전환. (예: KB증권 채용연계형)
[계약직 1~2년]             → 갱신 2회 한도, 이후 정규직 전환 검토. (근로기준법)
[수습 3개월]               → 거의 100% 본채용. 급여 80~90% 수습기간 감액.
[정규직 공채]              → 서류→인적성→1차→임원→최종. 바로 정규직 사원.
[전문계약직]               → 일반 계약직과 다름. 연봉제, 2년 상한 적용 안됨.
[파견근로자]                → 원청이 아닌 파견사 소속. 2년 초과 시 직접고용 의무 (파견법).
```

- **핵심 판별**: 공고 타이틀에 "채용연계형" / "전환형" 명시 ≠ "체험형" / "단기 인턴".
- 공공기관은 90% 체험형 (전환 불가). 민간 금융은 채용연계형 비율 증가 추세.

## 근로 조건 체크포인트

| 항목 | 법정 기준 | 금융권 실무 | 판정 |
|------|-----------|-----------|------|
| 주 근로시간 | 주 52시간 (40+12) | 은행 40~44h, 증권 IB 60h+ | >52h 명시 → 경고 |
| 연차휴가 | 1년 미만 월 1일, 1년+ 15일 | 초봉 15일, 근속시 최대 25일 | <15일 → 경고 |
| 육아휴직 | 1년 법정 | 은행 1~2년, 여성 90%+ 사용 | 명시 없음 → 중립 |
| 경조사비 | 없음 | 결혼/사망/출산 회사 규정 | 복리후생 가점 |
| 4대보험 | 의무 | 전부 가입 | 미가입 → 즉시 FAIL |
| 퇴직금 | 1년+ 근무시 의무 | DB/DC/IRP 중 하나 | 형태만 체크 |
| 주52시간 예외 | 일부 직군 | 트레이더/IB는 재량근로제 적용 가능 | 명시 필수 |
| 최저임금 | 2026 기준 월 210만원대 | 금융권은 전부 상회 | 미달 → FAIL |

## 자격증 매핑

한국 금융권에서 가산점 있는 자격증(가중치는 `kr/score.md`에서 계산):

**국가자격 (고용부/한국산업인력공단)**:
- ADsP (데이터분석 준전문가), SQLD (SQL 개발자) — 디지털/데이터 직무 기본
- 정보처리기사 — 금융 IT 직무 필수에 가까움
- 컴활 1급, 한국사능력검정 1~2급

**금융투자협회 (금투협)**:
- 금융투자분석사(금투사) — 증권사 리서치 필수
- 투자자산운용사(투운사) — 자산운용사 PM 진입 자격
- 신용분석사(신분사) — 은행 기업금융 리스크 심사
- 재무위험관리사(FRM 한국판) — 리스크관리 직무

**국제자격**:
- CFA (Level 1~3) — 글로벌 자산운용/리서치 프리미엄
- FRM (GARP) — 리스크관리 국제 표준
- CAIA — 대체투자 특화
- CPA (AICPA / KICPA) — 회계 기반 IB/리서치

**직무적합 우선순위 (찬희 타겟 기준)**:
1. 디지털자산: 없음 (아직 공식자격 없음) → 실적 기반
2. 핀테크: ADsP + SQLD + 정처기 콤보
3. 리서치: 금투사 + CFA Lv1
4. 은행 IT: 정처기 + ADsP

## 한국 금융권 조직도 맥락

공고의 부서명이 어디 계층에 속하는지 자동 파악:

```
[본점(Head Office)]
  ├─ 경영지원: HR, 재무, 총무, 법무, 홍보
  ├─ 리테일: WM(자산관리), 카드, 개인여신
  ├─ IB(기업금융): ECM, DCM, M&A, 구조화금융
  ├─ S&T: Trading, Sales, Structuring
  ├─ 리서치: 주식/채권/퀀트/이코노미스트
  ├─ 디지털: 플랫폼, 데이터, 블록체인, UX
  ├─ IT: 코어뱅킹, 보안, 인프라, DevOps
  └─ 관리: 리스크, 컴플라이언스, 내부감사, 준법감시

[영업점(Branch)]
  └─ PB, 개인금융, SME, 창구

[공공금융]
  └─ 한은, 금감원, 금융위, 예탁결제원, 거래소, 신보, 기보, 주금공, 예보, 캠코
```

- **찬희 타겟 매핑**: 디지털(블록체인/데이터) > 리서치 > IB-ECM > 리스크. 영업점은 회피.
- 공공금융은 체험형 인턴 위주 → 스펙용, 전환 기대 X.
