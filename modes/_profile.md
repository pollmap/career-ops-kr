## Core Narrative

나는 금융, AI, 블록체인이 만나는 교차점에서 실제로 동작하는 시스템을 만드는 사람이다. 충북대 경영학부 2학년까지 마치고 휴학한 뒤, Luxon AI를 창업해서 2개 VPS와 1개 WSL 위에 3개 에이전트와 MCP 64서버/398도구를 실운영 중이다. 동시에 CUFA(충북대 가치투자학회) 회장으로서 SMIC, HD건설기계, KSS해운, 플리토, JYP 같은 상장사 정밀 분석 보고서를 쌓아왔다. 내가 추구하는 건 "이론만 아는 사람"도 "코드만 치는 사람"도 아니다 — 기업을 뜯어보고, 데이터를 검증하고, 자동화 파이프라인까지 내 손으로 돌리는 사람이다.

## Superpowers

- **에이전틱 코딩 실전**: Claude Code + MCP 프로토콜로 64서버 398도구 프로덕션 운영. 동년배 기준 극히 드문 경험.
- **금융 데이터 허브 설계**: DART, KRX, KIS, 한은 ECOS를 MCP로 통합한 Nexus Finance 서버를 직접 만듦. CFS 전용, Forward/Trailing 기준 명시 같은 품질 원칙을 코드 레벨로 박아둠.
- **가치투자 리서치 산출력**: CUFA 회장으로서 80K자+ 리포트 파이프라인(v14.1 Evaluator v2 ALL PASS) 구축 및 5+ 기업 완성.
- **End-to-end 1인 인프라**: VPS systemd, nginx, UFW, SSH key-only, DNS까지 혼자 돌림. 장애 복구 프로토콜도 문서화됨.
- **실데이터 원칙**: 목업/할루시네이션 금지. 출처 표기 습관. "없다"고 명시하는 정직함.
- **문서화 습관**: Obsidian Vault(PARA) + SKILL.md 기반으로 지식을 자산화.

## Deal-breakers

- **영업/세일즈 직무**: 전화 영업, 신규 개척, 방문 판매는 맞지 않음.
- **단순 반복 사무**: 엑셀 타이핑만 하는 포지션은 성장 기회가 없음.
- **지방 상주**: 안산/서울 4호선권이 아니면 물리적으로 어려움.
- **졸업자 전용 공고**: 휴학 중 상태이므로 해당 없음.
- **경력 3년+ 요구**: 신입 + 프로젝트 경력만 있음.
- **허위/과장을 요구하는 자소서**: 실데이터 원칙에 반함.
